This application lets you track the EJO Star Wars Jedi Knight Jedi Academy server from your desktop.

It was made by me (ShadøwXPA), same guy who is making this... Somewhat questionable website... (I'm not good with design alright?).

Feel free to distribute this software, as long as you DO NOT profit from it.

Why Universal? Because it can be used on Windows, MacOS and Linux.
It was made with Java, which made that possible.
You will need Java 8 or higher to run the application.
If you need it, follow this link.

If you already have Java, simply double click the "EJOTrackerUni.jar" file.
Or if you'd like to run the Tracker from the command line, do the following:
Open the command line, and type:
javaw -jar "EJOTrackerUni.jar".
But before you do that, make sure you have unzipped all of the content.

This application asks the server for information every minute (you can change this in the configuration file, minimum time is 1 minute), this means, the tracker updates itself every minute.

When a player connects to the server, if the minute cooldown hasn't passed yet, it waits the remaining time, and once the tracker refreshes, it sends a notification (if allowed).

After the first notification, the tracker sends a new notification, every 30 minutes, with the number of players online (can be more or less players). This could be useful, if you're waiting for more players to join, or if you're waiting for players to leave the server, if the server is full.

If the real player's count drops to 0, the 30 minute cooldown resets. If there were some players on the server, and suddenly everyone left, it would send a notification again, when a new player joins (3 players [Notification, wait 30 minutes] -> 0 players [No notification, cooldown resets] -> 1 player [Notification, wait 30 minutes]).
