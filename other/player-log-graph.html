PlayerLog Graph

So you have used the PlayerLog feature from the tracker, and don't know what to do with those log files?
You've come to the right place...

This small application, which is only available on Windows, lets you view the server's activity from those logs.

Before you download this, I will let you know how to use this. It is pretty simple.

[1] - Menu
    This menu allows you to manage the application, since this is a fullscreen application, you can't minimize it or close it. So on the Window menu you can do that.
    On the File menu, you can open a .json file, and view the activity graph. You can also close the file from that menu.

[2] - Legend
    Tells you what the colors mean.

[3] - File Information
    You can view the number of snapshots the tracker recorded, the date of the first snapshot, and the date of the last snapshot, in UTC...

[4] - Graph
    In this part of the graph, there is no activity on the server, according to the log file.

[5] - More Graph
    And in this part of the graph, there is activity on the server.

On the Y axis you have the number of players. From that graph the maximum number of players recorded was 7, 4 of them were bots, which means 3 were real players.

On the X axis you have the snapshots.

This graph does not tell you the time of when there was activity, only the snapshots can tell you that. If you want to know the time you'll have to look for the snapshot(s) from inside the log file.

From the graph you can see activity around the end, so you'd look for the latest snapshots to find the time.

But, I'm sure you know how to read graphs! So no need to explain further.
